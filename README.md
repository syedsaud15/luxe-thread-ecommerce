# 👗 LUXE THREAD

A modern luxury fashion e-commerce website built with **Next.js, Node.js, Express.js, and REST API**.

## 🌐 Live Demo

Frontend (Vercel)

https://luxe-thread-ecommerce.vercel.app

Backend API (Render)

https://luxe-thread-backend.onrender.com/api/products

---

# ✨ Features

- Modern Luxury Fashion UI
- Hero Landing Page
- Product Catalog
- Category Filtering
- Product Search
- Shopping Cart
- Quantity Management
- Remove From Cart
- Responsive Design
- REST API Integration
- Express Backend
- Next.js App Router
- TypeScript Support

# 🚀 Installation

Clone the repository

```bash
git clone https://github.com/syedsaud15/luxe-thread-ecommerce.git
```

Frontend

```bash
cd frontend
npm install
npm run dev
```

Backend

```bash
cd backend
npm install
npm start
```
